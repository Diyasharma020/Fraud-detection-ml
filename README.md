# Credit Card Fraud Detection ML System

This project implements a fraud detection system using machine learning on an imbalanced dataset. The model was trained using SMOTE for imbalance handling and Random Forest for classification.

## Dataset
The dataset used in this project is publicly available on Kaggle:

https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud

Download the dataset and place `creditcard.csv` inside the `data/` folder before training.

## Model Performance
- Recall (Fraud Class): 0.83
- Precision (Fraud Class): 0.87
- ROC-AUC: 0.91
